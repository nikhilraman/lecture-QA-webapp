# lecture-QA-webapp

Web Application for a Piazza like Q/A service for in class help. Has some Reddit like features. 

Note: App uses DynamoDB on the backend and in order to test the application one should first run 'node loader.js' to initialize the database and then 'node app/js' to start the server. Valid AWS credentials are required in 'config.json' in order for DB communication to work.
